# Aeroric：把 AI 编程智能体真正放进你的桌面工作流

## 一句话介绍

Aeroric 是一个面向 AI 编程智能体的桌面工作台，把 Claude Code、Codex、自定义 Agent、多项目任务、实时终端、Git、SSH、SFTP、Docker、数据库、Markdown 文档、随手记和版本发布流程集中到一个轻量应用里。

## 推广标题

让 Claude Code 和 Codex 不再散落在终端里：用 Aeroric 管理你的 AI 编程工作流

## 内容介绍

如果你已经在用 Claude Code、Codex 或其他 AI 编程智能体，应该很熟悉这种工作状态：多个终端窗口同时跑任务，Git diff、项目文件、远程服务器、Docker、数据库、Markdown 文档和发布页面分散在不同工具里。AI 确实提升了编码效率，但工程上下文却越来越容易失控。

Aeroric 解决的就是这个问题。它不是替代 Claude Code 或 Codex，而是在它们外面加上一层更适合工程落地的桌面工作台。你可以在同一个界面里启动智能体任务、查看实时 PTY 终端输出、恢复历史会话、管理项目文件、阅读和编辑 Markdown、查看 Git 变更、连接 SSH/SFTP、检查 Docker 状态、操作数据库，并把版本发布流程留在项目上下文中完成。

对于经常同时维护多个项目、频繁使用 AI 生成和修改代码、需要远程排障或部署的开发者来说，Aeroric 可以显著减少工具切换，让 AI 编程从“临时终端会话”变成“可追踪、可恢复、可管理的工程流程”。

## 核心亮点

- **AI 编程任务集中管理**：统一运行 Claude Code、Codex 和自定义智能体，支持权限模式、实时输出、交互输入、会话恢复和任务取消。
- **多项目桌面工作台**：围绕项目组织任务、文件、Git 状态、终端和运行环境，减少上下文丢失。
- **IDE 级辅助能力**：支持文件浏览、代码编辑、Markdown 阅读/编辑、搜索替换、诊断查看、符号跳转、测试运行、DAP 调试和 Web 预览。
- **开发基础设施一体化**：集成 SSH、SFTP、Docker、端口管理和 DBX 驱动的数据库工具，适合本地开发、远程排障和部署检查。
- **发布流程更可控**：在项目上下文里完成 diff review、暂存、提交、推送和 Release 管理，降低发布时的工具切换成本。
- **长期任务可观测**：自动发现 Claude Code / Codex 会话，查看 token 消耗和工具调用指标，让长时间智能体任务更容易追踪。

## 适合谁使用

- 正在高频使用 Claude Code、Codex 或其他 AI 编程 Agent 的开发者。
- 同时维护多个本地或远程项目，需要统一管理任务、终端和文件上下文的工程师。
- 经常需要 SSH/SFTP、Docker、数据库检查、脚本运行和版本发布的全栈或后端开发者。
- 希望把 AI 编程流程产品化、规范化，而不是只依赖零散终端窗口的团队或个人。
